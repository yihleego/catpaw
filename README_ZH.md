# Cat Paw 🐾

[English](README.md) | [中文](README_ZH.md)

**Cat Paw** 是一个有趣、轻量级的桌面覆盖应用程序，使用 Rust 和 [Bevy](https://bevyengine.org/) 游戏引擎编写。
它用一个动画猫爪替换你的鼠标光标，猫爪会跟随你的移动并对你的点击做出反应！

## 功能特性

- **鼠标跟随**：猫爪会在屏幕上自然地跟随你的鼠标光标。
- **交互动画**：
    - **左键点击**：猫爪握紧（抓取）。
    - **右键点击**：猫爪张开手指（摊开）。
- **全局覆盖**：在覆盖整个屏幕的透明、置顶窗口中运行。
- **快捷键**：
    - **切换可见性**：同时按下 **左键 + 右键**（短按）可开启或关闭猫爪。当猫爪隐藏时，系统光标会恢复。
    - **退出**：长按 **左键 + 右键** 持续 **2秒** 可关闭应用程序。
- **跨平台**：专为 macOS 和 Windows 设计。

## 安装与构建

### 前置要求

你需要安装 **Rust** 和 **Cargo**。如果你还没有安装，请从 [rustup.rs](https://rustup.rs/) 获取。

### 开发

要在开发模式下运行应用程序：

```bash
car go run
```

### macOS 构建

要为 macOS 创建独立的 `.app` 包：

1. 在项目根目录打开终端。
2. 运行构建脚本：
   ```bash
   ./build_macos.sh
   ```
3. 应用程序将构建在 `target/release/bundle/CatPaw.app`。
4. 你可以使用 `open target/release/bundle/CatPaw.app` 打开它，或者将其拖到你的应用程序文件夹中。

### Windows 构建

要为 Windows 创建分发文件夹：

1. 在项目根目录打开命令提示符或 PowerShell。
2. 运行构建脚本：
   ```cmd
   build_windows.bat
   ```
3. 应用程序将位于 `target\release\distribution\CatPaw`。
4. 运行该文件夹内的 `CatPaw.exe`。

## 使用方法

应用程序运行后：

- **移动鼠标**：猫爪跟随移动。
- **左键点击**：观看猫爪握紧。
- **右键点击**：观看猫爪张开。
- **隐藏/显示**：快速同时按下 左键+右键。
- **退出**：同时长按 左键+右键 2秒钟。

## 已知问题

- **Vulkan 窗口透明度**：在某些使用 Vulkan 图形 API 的环境中，窗口透明度可能会失效，导致背景显示为黑色而不是透明覆盖层。这通常与
  GPU 驱动程序支持或系统合成器设置有关。
